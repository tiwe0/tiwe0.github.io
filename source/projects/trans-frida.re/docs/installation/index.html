安装

安装Frida并配置好使用只需要几分钟。如果你遇到了什么麻烦，请 提交issue(或者pr)来描述你遇到的问题。

Frida’s CLI 工具的依赖

安装 Frida’s CLI 工具非常简单直接，但是你还是有一些依赖。

  • Python – 强烈推荐最新的 3.x
  • Windows, macOS, or GNU/Linux
