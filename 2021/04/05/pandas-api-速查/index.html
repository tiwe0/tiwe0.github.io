1.1.3 DONE 对 Pandas 的理解

  1. pandas 的操作只有两类
    • 返回数据
    • 暴露接口

    pandas 不会处理数据，而是提供以上的操作，由用户自己指定处理。

  2. 视图 与 拷贝

    在默认情况下，pandas 中的绝大多数操作均没有副作用，均不会对原数据有所影响。

    • 视图意味着返回了内存切片
    • 副本意味着返回了数据拷贝
  3. 在 pandas 里，数据的结构与数学中是对应的。
    数学 pandas
    数据矩阵由若干个维度一样的列向量构成 DataFrame 由若干个长度一样的 Series 构成
    数据矩阵的行表示观测，列表示数据维度 DataFrame 的行为观测，列为维度
  4. 当从数据框中选择数据时：
    • 若为向量,则统一以 Series 形式呈现,可以调用 Series 的 API；
    • 若为矩阵，则统一以 Dataframe 形式呈现，可以调用 DataFrame 的 API。
  5. 筛选

    不管底层是如何实现的，在本文档中，均按照

    1. 生成 bool 掩码。
    2. 根据 bool 掩码筛选数据

    的思路进行

1.1.4 DONE 名词区分

  1. 筛选 与 选择
    • 筛选:选取数据框
    • 选择：选取观测或者向量（一维）
  2. 向量 与 观测
    • 在前面的约定中，向量是列，观测是行。
      • 从数学上看，观测也是向量。
      • 在 pandas 中，两者都以 Series 形式实现。
    • 向量一词往往会根据语境发生变化。
  3. 暴露 与 遮掩

    在筛选中：

    • 暴露：被暴露在结果中，不会被筛选掉（或被替换掉）
    • 遮掩：被遮掩掉，余下的空位会被筛选掉（或被替换掉）
    • 当说到暴露时，默认余下部分被遮掩；反之亦然。
  4. 暴露接口 与 暴露
    • 暴露接口：操作的接口指被暴露出来作为句柄（被拿来操作）
    • 暴露：暴露出来作为结果（因此不会被改变）
  5. 数据维度 与 维度
    • 维度：指数学意义的维度
    • 数据维度：指二维数据的数据框的列数

1.1.5 DONE Misc

  • 返回某个对象时，默认返回与原对象形状，维度一致的某种类型的对象
  • 同上：是指两个函数完全相同，互为别名的关系
  • 沿着某个方向：指某个方向为形成的向量为整体
  • 以下叙述中“返回”会出现很多次，在不引起误解的情况下，会作以省略

1.1.6 TODO 记号：

  • []：可有可无的参数
  • func：函数引用
  • axis：维度方向： 从 0 到 n-1； n 为数据的维度，在常见的二维数据中，0是沿着向量的方向

2.1.6 Apply & GroupBy & window

  1. Apply

    transform 与 apply 的区别：

    1. transform 不能用于聚合数据；而 apply 可以 transform 作用的是传入数据的元素，而 apply 作用的是传入数据的整体
    2. transform 可以传入单个函数，字符表示的函数，函数组成的字典；而 apply 只能传入单个函数（而且必须是引用）
    3. transform 一次只能作用在一个向量之上；而 apply 可以同时作用多个向量
    函数 解释
    Series.apply(func[,convertdtype, args]) Apply
    Series.agg([func, axis]) 聚合
    Series.aggregate([func, axis]) 同上
    Series.transform(func[, axis]) 转换
    Series.map(arg[, naaction]) 对 Series 映射
  2. GroupBy
    函数 解释
    Series.GroupBy([by, axis, level, asindex,…]) 根据某种条件进行分组，返回一个分组对象
  3. window
    函数 解释
    Series.rolling(window[, minperiods,…]) 同步窗口运算
    Series.expanding([minperiods,center,axis]) 异步窗口运算
    Series.ewm([com, span, halflige, alpha,…])  
    Series.pipe(func, *args, **kwaargs) 类似 unix 下的 pipe

2.2.6 Apply & GroupBy & window

transform 与 apply 的区别：

  1. transform 不能用于聚合数据；而 apply 可以 transform 作用的是传入数据的元素，而 apply 作用的是传入数据的整体
  2. transform 可以传入单个函数，字符表示的函数，函数组成的字典；而 apply 只能传入单个函数（而且必须是引用）
  3. transform 一次只能作用在一个向量之上；而 apply 可以同时作用多个向量
  1. DONE Apply

    DataFrame.apply(func[, axis, raw, …]) 应用，默认传入的是指定方向的向量
    DataFrame.applymap(func[, naaction]) 应用，默认传入的是元素
    DataFrame.pipe(func, *args, **kwargs) 类似 unix 下的 pipe;类似 R 中的 %>%；类似 JavaScript 中的连续调用#TODO
  2. DONE GroupBy
    函数 解释
    DataFrame.agg([func, axis]) 聚合
    DataFrame.aggregate([func, axis]) 同上
    DataFrame.transform(func[, axis]) 转换
    DataFrame.groupby([by, axis, level, …]) 分组
  3. TODO window
